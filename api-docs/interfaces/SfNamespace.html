Interface SfNamespace

Represents the namespace within Sitefinity that contains rich text editor tools.

The SfNamespace interface includes a dictionary of tools (rteTools) and a callback function (rteToolsIconsSetupCallback) that is responsible for setting up tool icons in the editor's UI.
