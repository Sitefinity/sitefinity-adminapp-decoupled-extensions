Interface DataItem

Represents a data item in Sitefinity, including its actual content, metadata, provider, and culture information.

A DataItem encapsulates the core content along with additional metadata that provides context about the content's structure, the data provider, and the associated culture. This information is crucial for tools that need to interact with the content in a structured manner.

interface DataItem {
    culture: string;
    data: any;
    key: string;
    metadata: Entity;
    provider: string;
    title: string;
}
